首先，思考自己的人生，思考自己想要做什麼 ，並列出清單。接著列出所有要做的事情，在小的筆記本上標示時刻，以及預定要做的事情，剩下的時刻填上今天要做的事情，如果有沒安排到的事情，就移到其他日期。

想像一下，如果你有存款一億美金，像我會去日本旅遊、買房子、買車子，買一堆東西，等到該買的該玩的都做了，然後呢？日常生活會做什麼？如果你不再需要為錢工作，你平常會做什麼？好好想想並寫下來，這對時間管理至關重要，很多人無法回答這個問題，所以時間管理失敗了。規劃好時間表卻沒有照著做，因為不知為何而戰；好不容易下班了，休息時間只會被動的看 YouTube 和 Netflix，因為不知道主動做什麼想做的事。

列出所有要做的事情，不要用腦子記，可以用 Google 日曆之類的，或是紙本行事曆，然後在另外一個小的橫線筆記本上標示時刻，例如像這樣代表早上 8 點到下午 2 點：

| 8 |    |
| :-- | -- |
| 9 |     |
| 10 |     |
| 11 |     |
| 12 | 午餐 |
| 1 |     |
| 2 |     |

填上預定要做的事情，最基本的是早午晚餐、洗澡和睡覺，另外要加上想做的事，剩下的時刻填上需要做的工作事項，最後沒有安排的事項移到明天或其他日期。

重要的事情自然會先做，否則它就不重要。先把事情分的很細？但是事情通常做到一半才知道有哪些步驟，沒辦法事先拆分清楚。
