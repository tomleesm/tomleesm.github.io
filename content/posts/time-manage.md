以下是我覺得有用的時間管理方法

一、記錄做了什麼和花了多少時間

參考[柳比歇夫時間管理法](https://bailushuyuan.org/%E5%A5%87%E7%89%B9%E7%9A%84%E4%B8%80%E7%94%9F)，不想看書的話可以看這個[懶人包影片](https://www.youtube.com/watch?v=-EbFcCV-Kbo)。我覺得他這個方法其實是在記錄心流時間，所以只記錄重要的工作耗費多久，沒有記錄吃飯、洗澡或午睡，雖然這些時間花費好幾個小時（一天 24 小時減掉工作 6 小時和睡覺 10 小時，剩下 8 小時）。

只有處在心流時間，才會有效率，工作才會有進展。而記錄每天的心流時間，幫助自己留意它、在乎它、進而想辦法增加心流時間。

二、優先做自己喜歡的事情

來自於[EP211 哪種 #時間管理 方法最有效？不花你太多時間的那種！](https://www.youtube.com/watch?v=rDXJuSiC5Jk)。每天早上先做自己喜歡的、真正想做的事情，而不是必須要做的事情。

三、事先規劃好等候時可以做什麼

需要等候的時間其實很多，如果沒有事先規劃，就會覺得應該沒有那麼多等候時間吧？然後用來發呆或滑手機（看影片或新聞）。

| 需要等候的事          | 等候時可順便做的事 |
| --------------- | --------- |
| 1. 煮開水          | 1. 洗鍋子、餐具 |
| 2. 洗衣服          | 2. 看小說    |
| 3. 下載檔案，或上傳     | 3. 刷洗浴室   |
| 4. 安裝程式，或更新作業系統 | 4. 刮鬍子    |
| 5. 等公車或捷運       | 5. 剪指甲    |
| 6. 等水滾或燉煮食物     | 6. 掃地     |
|                 | 7. 外出買東西  |
|                 | 8. 背單字    |
|                 | 9. 速寫     |
